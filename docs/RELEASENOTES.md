# ShopnoOS-beta (March 25, 2024)
---
### System Enhancements

- Implemented ShopnoOS exclusive backgrounds for both GRUB bootloader and LightDM display manager for a distinct visual identity.
- Modified the selection accent color that is more aligned with JaduPc within the Calamares installer for improved visual coherence.
- Introduced new slide-backgrounds within the Calamares installer to enhance the installation experience.
- Integrated ShopnoOS backgrounds across the system, including the default desktop background.
- Resolved an issue where default backgrounds were erroneously set to Debian defaults on portable devices, ensuring consistency.
- Configured a default panel layout with the Window Button applet for enhanced usability.

### Live Environment Improvements

- Updated the hostname of the live environment to ShopnoOS-Live.
- Standardized the live environment username to `live-user`.

### Software and Package Updates

- Included GNOME Software Center for application discovery and management.
- Included Noto fonts to expand font options and support multilingual text rendering.
- Addressed broken Bengali font rendering in Chromium for improved web browsing experience.
- Implemented a modified chromium.desktop file to mitigate keyring password prompts.
- Integrated the rtl8821ce driver for improved compatibility with specific network hardware.
- Pre-installed the LAMP stack, streamlining web development and server deployment processes.
- Replaced KCalc with Galculator for GTK consistency.

### Additional Features

- Added third-party repositories for VS Code and Sublime Text, enabling easy access to popular development tools.
